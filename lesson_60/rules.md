# Lesson 60

## Summary
We learn the "we" form in the point-in-the-past. Bonus: *-ir* verbs use *-imos* for both present AND past, so *vivimos* / *salimos* / *perdimos* are the same in both — context tells which is which.

## Rules
1. Recap: I spoke = *hablé*; he spoke = *habló*; I ate = *comí*; he ate = *comió*.
2. "We spoke" in the point-in-past = *hablamos*. Wait — that's the same as the present "we speak"!
3. So *hablamos* = "we speak" OR "we spoke". Spanish doesn't worry — context handles it.
4. Same trick for *-ir*: *vivimos* = "we live" AND "we lived" (no change).
5. BUT *-er* verbs DO change: present *we eat* = *comemos*, past *we ate* = *comimos*. Different ending!
6. Why? Remember the one rule that *-er* and *-ir* split only at the "we" form in the present (*-emos* vs *-imos*). When we go to past, both use *-imos* — so *-ir* matches its own present, but *-er* now differs.
7. Examples to lock in:
   - *vemos* (we see, present) → *vimos* (we saw, past).
   - *salimos* = "we leave / go out" AND "we left / went out" — same.
   - *perdemos* (we lose) → *perdimos* (we lost). Different.
   - *vivimos* = "we live" AND "we lived" — same.
8. Use context words (*ayer*, *cuando*, *mañana*) to make tense clear when the form is ambiguous.
9. Spanish often uses the present with future context: *Hablamos mañana* = "We will speak tomorrow".
10. *Cuando hablamos, comprendí todo* = "When we spoke, I understood everything" — the second half tells you *hablamos* is past here.
11. *Esperar* = to wait OR to hope. Mnemonic: "desperate" = *desesperado* in Spanish — "un-hoped". You also see *desesperar* (to despair).
12. Watch out for fake-Spanish in English. We say "desperado" but the real Spanish is *desesperado*. We say "no problemo" — it should be *ningún problema* (and *problema* is the word, not "problemo").
13. *Mucho* = a lot.
14. *Nos perdimos* = "we got lost" (we lost ourselves) — using *se* / *nos* reflexively.
15. *Siempre* = always. Sentence position is flexible: *Siempre nos perdemos* / *Nos perdemos siempre*.
16. *A menudo* = often. Mnemonic: *menudo* literally means "small" (like "minute" / "miniscule"), but as an expression *a menudo* means "often". You won't hear *menudo* alone for "small" much.

## Vocabulary
- *hablar / comer / vivir* — to speak / eat / live
- *hablé / habló / hablamos* — I/he/we spoke
- *comí / comió / comimos* — I/he/we ate
- *viví / vivió / vivimos* — I/he/we lived (vivimos = both present and past)
- *cuando* — when
- *comprender* — to understand
- *comprendo / comprendí* — I understand / I understood
- *todo* — everything
- *ayer* — yesterday
- *mañana* — tomorrow
- *mucho* — a lot
- *esperar* — to wait / to hope
- *esperamos* — we wait / we waited
- *desesperar* — to despair
- *desesperado* — desperate ("un-hoped")
- *problema* — problem (mas. ending in -a)
- *ningún problema* — no problem
- *ver* — to see
- *vemos / vimos* — we see / we saw
- *salir* — to leave / go out
- *salimos* — we leave / we left (same)
- *perder* — to lose
- *perdemos / perdimos* — we lose / we lost
- *perderse* — to get lost
- *nos perdemos / nos perdimos* — we get lost / we got lost
- *siempre* — always
- *pero* — but
- *a menudo* — often
- *menudo* — small (literary; not used much)

## Examples / Sentences
- *Hablé. / Habló. / Hablamos.* — I / he / we spoke.
- *Comí. / Comió. / Comimos.* — I / he / we ate.
- *Cuando hablamos, comprendo todo.* — When we speak, I understand everything.
- *Cuando hablamos, comprendí todo.* — When we spoke, I understood everything.
- *Hablamos ayer.* — We spoke yesterday.
- *Hablamos mañana.* — We'll speak tomorrow (present with future context).
- *Esperamos mucho.* — We wait a lot / We waited a lot.
- *Esperamos mucho ayer.* — We waited a lot yesterday.
- *Comemos / Comimos.* — We eat / We ate.
- *No comimos.* — We didn't eat.
- *Vivimos.* — We live / we lived.
- *Vemos / Vimos.* — We see / we saw.
- *Salimos.* — We leave / we left.
- *Perdemos / Perdimos.* — We lose / we lost.
- *Nos perdimos.* — We got lost.
- *Siempre nos perdemos.* — We always get lost.
- *No nos perdimos ayer.* — We didn't get lost yesterday.
- *No salimos a menudo, pero salimos ayer.* — We don't go out often, but we went out yesterday.
