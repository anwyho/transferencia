# Cross-References — Language Transfer Spanish (90 lessons)

This is the map of how the lessons connect. Read it like a spider web: each lesson hangs on threads from earlier ones, and each one usually tugs on a thread that gets extended later.

How to use:

- **Theme threads**: follow one idea (e.g. "the three pasts", "mood tense") across all the lessons that touch it.
- **Per-lesson back-links**: a quick lookup of which earlier lessons each lesson directly builds on.
- **Forward links**: where a small idea planted early gets fully bloomed later.

Each lesson directory `lesson_NN/rules.md` already has its own `## Cross-references` section listing references the teacher said out loud — this document is the bird's-eye view that joins them all.

## Theme threads

### 1. Pronunciation & spelling

L2 (5 pure vowels, *es*/*no es*, stress is part of the word) → **L3** (Latin → Spanish: *-al*, *-ant/-ent → -ante/-ente*, *-ly → -mente*, English /j/ → Spanish /kh/) → **L5** (single *r* relaxed; double *rr* rolled) → **L6** (silent *h*; *v* sounds like soft *b*) → **L9** (q-u always /k/) → **L25** (q-u recap) → **L31** (Spanish can't start *st-/sp-* → adds *e-*: *España, espagueti*) → **L40** (*ñ* letter; the *gn* → *ñ* historical fact) → **L55** (F → H sound history: *facere → hacer*) → **L67** (full written-accent rules: vowel/n/s = penultimate stress; everything else = last; written accent only when this rule is broken; monosyllable disambiguation: *sí/si, tú/tu, mí/mi, sé/se*) → **L72** (commands and added accents when little words attach: *espérenme, cómpralo*) → **L88** (spelling switch *c→z* in *hizo*; dropped *i* in *fueron/dijeron*) → **L90** (regional accents and aspirated *s*).

### 2. The convertible-words trick (free vocabulary)

L2 (*-al* words come over unchanged: *natural, animal, total*) → **L3** (*-ant/-ent → -ante/-ente*, *-ible/-able* same, *-ly → -mente*) → **L4** (THE BIG ONE: *-ation → -ación*, then chop *-ación* → *-ar* and you have a verb. Around 1,000 verbs unlocked: *cancelación → cancelar*) → **L5** (*-ence/-ance → -encia/-ancia*) → **L9** (*explicación → explicar* exception that proves the rule) → **L12** (*-tion → -ción*, *-ity → -idad*) → **L29** (*-idad* still in play) → **L33** (*-idad / -ity*) → **L41** (*-ción* and /j/→/kh/ recap) → **L70** (*-ity → -idad*) → **L79** (*-tial → -cial*).

### 3. The "spine" verbs (helper verbs)

L4 *quiero* → **L6** *intento* → **L7** *voy a + verb* → **L8** *quiere*, *debo* → **L9** *saber* → **L10** *tener* (and the *-tain* family: contain, retain, maintain) → **L11** *deber*, *voy a* recap → **L13** *hacer / poder / esperar* → **L14** the spine roll-call: *quiero / voy a / tengo que / puedo / debo* → **L21** *he hecho*, *tengo que* contrasts with *he* → **L26** *quedar* with all helpers → **L40** spine recap → **L55** *deber* + would → **L74** *volver* in spine.

### 4. The "I" form (-o ending) and *-go* irregulars

L6 *intento, voy* → **L10** chop *-r*, add *-o*, stress on penultimate: *hablo, como* → **L11** confirms across the three groups → **L12** full *-ar* present → **L13** *hago, puedo, espero, tengo, vengo, salgo* (the *-go* family) → **L14** more *-go* (*pongo, supongo, compongo*) and *sé* (*saber*) as a peculiar I-form → **L24** *doy* (-oy ending: *voy, doy, estoy*) → **L29** *estoy* → **L31** *-oy* family recap → **L57** *vengo* irregularity → **L84** the full "go-verbs" family (*tengo, vengo, salgo, pongo, digo, hago, oigo, traigo*) → **L86** the same family contracts in the future (*tendré, saldré, vendré, pondré*).

### 5. *Voy a* future and the slow-build to a real future tense

L7 *voy a + verb* (informal future) → **L19** *vamos*, irregular *ir* → **L20** future via context (present + time word) → **L26** *quedar* with *voy a* → **L36** real future tense: infinitive + *-é, -ás, -á, -emos, -án* (haber-endings shoved onto the full verb) → **L41** future + conditional together → **L42** *poder → podría* contracts → **L86** "go verbs" contract their future (*tendré, vendré, pondré, saldré*); *hacer/decir* extra irregular (*haré, diré*); *poder → podré*.

### 6. Conditional ("would")

L41 *-ía, -ías, -íamos, -ían* on the full verb (mirror of the future) → **L42** *me gustaría*, *podría* → **L43** *habría* + past participle = "would have"; *debería* = should → **L44** *podría / debería* practice → **L46** *habría* re-introduced as line of past haber → **L48** *-ía* sound serving double duty (line-past *-er/-ir* and conditional, distinguished by where it attaches) → **L50** *me gustaría* as the canonical hook → **L53** *podía* (line) vs *podría* (would) — the easy-to-mix-up pair → **L83** *si pudiera... lo cancelaría* (past mood + would).

### 7. Object pronouns / little words ("things before the verb")

L4 *me, te* → **L5** *me, te, lo* → **L7** *los* → **L8** *la, los, las* → **L11** placement: before the changed verb, OR attached to a "to" form → **L12** stacking (*me lo venden*) → **L21** little words before the verb (with *haber*) → **L22** subject vs object pronouns mapped → **L24** *dar* + stacking → **L25** *me/te/nos* extended to "to/for me" → **L34** placement on *-ando/-iendo* gerunds → **L37** *lo/la/los/las* → *le/les* when the meaning is "to/for him/her" → **L38** reflexive *se*; attaching to infinitives (*-se*) → **L43** ambiguity of *le/les* and the *con él/ella* dodge → **L62** L-L phonetic fix: *le lo → se lo* → **L63** L-L recap + clarifying *a* → **L64** *le → se before lo/la* recap → **L70** *se lo des* → **L72** little words attach to the END of positive commands, but stay BEFORE negatives → **L80** all uses of *se* (*sé*, self-*se*, phonetic *se*, on-me *se*) → **L81** *se me* + mood tense.

### 8. Reflexive *se* / *me* / *te* / *nos*

L23 *me preparo* (Spanish uses "myself" where English wouldn't) → **L25** *me/te/nos* extended to "to/for me/us" → **L38** *se* introduced fully = "self"; reflexive verbs end in *-se* in dictionary → **L40** *enamorarse de* (need a preposition) → **L47** *se* for "itself / get" (*la cena se quemaba* — passive feel) → **L48** *se vende*, *¿cómo se dice?* → **L60** got-lost *nos perdimos* → **L61** *quedarse* with point/line past → **L62** phonetic *se* (different from reflexive *se*) → **L70** reflexive verbs after mood triggers → **L74** *romperse, encontrarse* (leave doer unnamed) → **L80** ALL uses of *se* in one place → **L81** *se me cierra la puerta* + mood.

### 9. Gender, number, and agreement

L8 *lo/la, los/las* → **L12** feminine endings *-a, -ión, -idad* → **L19** *-a, -ión, -idad* recap → **L21** *-ión* feminine again (*celebración*) → **L23** *-ma* exception: Greek-origin words are masculine (*el problema, el sistema*) → **L27** adjectives go AFTER nouns and agree → **L28** full noun + adjective agreement (*-o/-a* singular; *-os/-as*, *-es* plural; double-letter rule); *muy*; *son* → **L44** more gender exceptions: a few *-o* words feminine (*la mano, la foto, la moto*) → **L54** feminine examples (*idea, película, persona, silla*) → **L65** possessives (*mi/tu/su/nuestro*) agree in number → **L66** skipping possessives for body parts (*me duele la pierna*) → **L89** short feminine *a-...-a* words take *el* (*el agua fría*); *uno/alguno/ninguno → un/algún/ningún* before masc. nouns; *demasiado* changes with nouns but not adjectives.

### 10. *Ser* vs *estar* (the two "to be"s)

L29 introduces the split: *ser* = characteristics, *estar* = states; full *estar* with *-oy* and final-syllable accent (*estás, está, están, estamos*) → **L30** *cómo estás*, full subject pronouns (*yo, tú, él, ella, nosotros, ellos, ellas*); *tener que* / *deber* recap → **L31** full *ser* (*soy, eres, es, somos, son*); *st-/sp- → est-/esp-*; *ustedes* → **L32** adjectives that change meaning with *ser* vs *estar* (*aburrido* = boring/bored) → **L33** more flexible adjectives (*listo* = ready/smart, *bueno* = good/handsome, *feliz*) → **L35** *estar siendo* (being X) for characteristics in progress → **L51** *estar para/por ti* → **L54** *-ado/-ido* participles as adjectives pair with *estar* → **L55** *estar* with *abierto, cerrado, roto* → **L82** *ser/estar + adj + que* → mood tense.

### 11. The gerund (*-ando / -iendo*) — Spanish "right now" *-ing*

L34 introduces *-ando/-iendo* for THIS-MOMENT-ONLY actions; placement of object pronouns on the gerund → **L35** *estar siendo*, *salir/salgo*; resist English's *-ing* overuse → **L40** *estar* + *-ando* for "right now" recap → **L44** *-ando/-iendo* recap → **L51** "no *-ing* after prepositions" rule (*gracias por venir*, NOT *viniendo*) → **L52** *-ando/-iendo* with *estar*; *-ante/-iente* for *-ing* adjectives (*interesante*); days of week from planets → **L57** *queriendo* recap → **L78** *seguir* + *-ando/-iendo*; *empezar a* + INFINITIVE (NOT gerund); *terminar de* vs *terminar* + gerund.

### 12. Stem-changing verbs (e→ie, o→ue, e→i)

L15 the rule: stressed root *e → ie*, *o → ue*; *querer/quiero, tener/tiene, pensar/pienso, entender/entiendo, perder/pierdo*; *tempo → tiempo* shows the same split between English and Spanish → **L16** more splits: *empezar/empiezo, poder/puedo, dormir/duermo, encontrar/encuentro, mostrar/muestro*; reflexive *me duermo* → **L17** splits don't ALWAYS happen (*comer*); the *-amos* "we" form moves stress off the root, so no split there → **L18** *-emos* vs *-imos*; same split rule applies → **L20** *vene → viene* recap → **L21** *tempo → tiempo* recap → **L26** *pensar → pienso* recap → **L27** *o → ue* in *encuentro, puedo* → **L57** the one *-er* vs *-ir* split (*-emos* vs *-imos*) → **L60** *we* point-in-past: *-ar* and *-ir* match the present, but *-er* differs (*comemos* vs *comimos*) → **L63** *acordarse* (*o → ue*) → **L64** the *e/i* flip between present and past for *-er/-ir* verbs → **L66** *doler → duele* → **L71** *contar → cuento* → **L74** *elegir* (e → i — the third pattern) → **L75** *seguir* family (e → i) → **L78** *seguir* (e → i) recap → **L80** *cerrar (cierra)* → **L81** *cerrar* mood-tense → **L90** *vos podés / querés* — vos doesn't split the vowel.

### 13. The three pasts (the most layered thread in the course)

**Have-past (perfect)**: L21 *haber* (*he, has, ha, hemos, han*) + *-ado* participle for "have eaten" with *-ar* verbs → **L22** *-er/-ir* past participle = *-ido* → **L23** reflexive *me he perdido* → **L26** recap → **L27** recap → **L43** *habría* + participle = "would have" → **L46** the three-pasts overview, line-past intro, *habría* recap → **L53** *había* + participle = "had done" (pluperfect).

**Line-past (imperfect, "the line in the past")**: L46 *-aba* for *-ar* → **L47** more *-aba*, *se* for "got" → **L48** *-ía* for *-er/-ir* (and the same *-ía* sound pulling double duty for "would" — distinguish by where it attaches) → **L49** three irregular line-pasts: *veía, era, iba* → **L52** *-aba/-ía* recap → **L53** *podía* (line) vs *podría* (would) → **L56** *iba* = irregular *ir* line-past → **L66** *tenía, dolía* → **L68** *estaba, dormía* → **L82** *era* via "era" mnemonic.

**Point-past (preterite, "the dot in the past")**: L58 *-ar* point-past: *-é / -ó* with critical accents; passive *se envió* → **L59** *-er/-ir* point-past: *-í / -ió*; double negatives → **L60** "we" point-past: *-ar* and *-ir* match present (*hablamos, vivimos*), *-er* differs (*comimos*) → **L61** line + point together (was-cooking-when-rang) → **L62** *dar* (*di / dio / dimos*) — note: *dar* ironically takes *-ir* endings → **L64** *hace* + time = "ago"; you-form *-aste/-iste*; they-form *-aron/-ieron* → **L65** point-past of *ver* → **L68** *visité, visitaron* → **L83** point-past as the source of the past mood tense's stem → **L87** line-past irregulars (era/iba/veía) + irregular point-past family with three traits (root change, mixed *-ar/-er-ir* endings, no accent): *tuve, estuve, pude, hube* → **L88** more irregular point-past (*puse, hice/hizo, vine, traje, fui*); *fui* shared for *ser* AND *ir*; spelling *c→z* in *hizo*; dropped *i* in *fueron/dijeron*.

### 14. Mood tense (the subjunctive)

L69 first introduction: take the *I*-form, swap groups (*-ar* takes *-er/-ir* endings, and vice versa); obligatory after *quiero que* → **L70** *lo que*, more triggers (*necesito que, prefiero que, hace falta que*); *faltar/falta* → **L71** *espero que, me gusta que*; *contar/contar con*; mood-tense of *ver* built from *veo* (→ *vea/vean*); the personal *a*; *a + el = al* → **L72** commands: mood for ALL negative commands, formal positive, plural positive, and *let's*. The one shortcut: positive informal = drop the *s* from the *tú* form. Little words attach to end of positive commands, before negative → **L73** mood for "let's", "let him/her", *cuando* + future event; mood-tense of *ir* (*vaya*) → **L74** *elegir* (e→i); *para que* + mood; irregular *hecho* after *haber*; *también* breakdown → **L75** *donde/como/que/cuando + mood = wherever/however/whatever/whenever*; irregular *hacer* command *haz*; *seguir* family; mood-tense of *ser* (*sea*) → **L76** *hay/había* (never plural); mood for questioning existence; *tiene que haber* → **L77** "es + adj + que + mood" (*es terrible que, es importante que*); *qué fuerte que...* → **L78** *seguir* + -ing; *empezar/comenzar a* vs *terminar de* → **L79** *dejar*; *cuando* + future review; *es esencial que*; *ojalá* (Arabic origin) + mood → **L80** all uses of *se* + the *se me / se te* on-me structure → **L81** *se me* + mood; *volver a* = "again"; PAST MOOD TENSE (*hablaran/hablara/habláramos*); *quería que / esperaba que / necesitábamos que* + past mood → **L82** mood after *ser/estar + adj + que*; the *lo + adj + es que* trick to AVOID the mood; *era* line-past of *ser* → **L83** past mood after *que* and after *si* (hypothetical past); alternative *-ase/-iese* form → **L84** "go verbs" mood swaps *-go* for *-ga* across all persons (*tenga, venga, salga, ponga, diga, haga, oiga, traiga*) → **L85** go-verb commands: formal uses *-ga*; informal positive command shortcut = drop the *-go* (*ten, sal, ven, di, pon*); *cuando* + future = mood → **L88** building past mood from the *they*-form of point-in-past (*fueran, tuvieran, hicieran*).

### 15. Para vs por (the two "for"s)

L50 *para* (destined for) vs *por* (because of / time periods / exchange); *día* is masc → **L51** *estar para/por ti*; "no *-ing* after prepositions" rule; false friends *discutir, actual* → **L52** *para = "in order to"*; days of week from planets → **L62** *para mí* (no repetition with *para*) → **L74** *por mí* vs *para que* + mood.

### 16. Prepositions in general

L39 *con, para, de*; after a preposition use *él/ella/ellos*, EXCEPT *mí* and *ti*; *le/les* as "imaginary preposition"; Spanish drops prepositions English keeps → **L40** preposition swaps between languages: *enamorarse de* (NOT "enamour with"), *pensar en* (NOT "of"), *soñar con* (NOT "of"); *conmigo, contigo* → **L50** *para vs por* → **L51** "no *-ing* after prepositions" → **L52** *para* = in order to → **L56** prepositions never end a sentence; *que/cual/quien* relatives.

### 17. Possessives, demonstratives, "this/that"

L47 *este/ese, estos/esos* → **L50** *este/esta/estos/estas* recap → **L51** demonstrative pattern → **L65** *mi/tu/su/nuestro*; *propio* = "own" → **L66** Spanish skips possessives for body parts (*me duele la pierna*); *la casa de X* = "X's house" → **L68** *este/ese/esto* family (this/that, including neuter); *tan*; *creer* → **L83** *mi casa, tu casa, su casa* recap.

### 18. Question words and *qué/cual*

L8 *por qué* → **L9** *sí* vs *si* → **L20** *qué* / *cómo* / *dónde* → **L25** *cuándo, cuánto* → **L42** *que* vs *qué* → **L56** *qué/cuál/quién*; *cuál* and *quién/quiénes* questions → **L67** question-word accents (*dónde, cómo, cuándo, cuánto*) → **L68** *cuántas veces*.

### 19. The "course method" (process > content)

L1 the thinking method: no memorizing, no writing, pause and answer for yourself, take it slow → **L4** positive vs negative language transfer (Latin and shared roots help; assuming English structures hurts) → **L9** stitching long sentences block by block → **L17** going back to the "to" form to avoid Turkish-style vowel harmony → **L33** ser/estar as a tool, not an obstacle → **L41** "se trick" for English "get + verb" → **L51** keep thinking sentences through piece by piece (recurring) → **L52** "we form often escapes irregularity" (recurring) → **L53** explicit forward reference to "dot in the past" → **L57** build forms directly without detours → **L74** mood tense as Spanish "value system" → **L76** mood-tense to express uncertainty → **L82** *lo + adj + es que* trick to AVOID the mood when stressed → **L89** rule-discovery lesson: three puzzles you solve yourself → **L90** dialect variations (leísmo, vosotros, vos, aspirated *s*) and the closing pep talk on listening as a separate skill.

### 20. Verbs whose I form is the noun-shape, and the "noun ↔ verb" pun

L13 *hago / un hecho*; *espero / la espera* → **L14** *pongo / una posición*; *supongo / una suposición* → **L16** *un encuentro / encuentro*; *una muestra / muestro* → **L29** *ser → un ser* → **L37** *cocinar* from *cocina* → **L74** *hecho* irregular past participle.

### 21. *Gustar*-style "back-to-front" verbs

L42 *gustar* = "to be pleasing" (don't say *me gusto*!); always start with the verb → **L43** *gustar* "Spanglish step" → **L45** *interesar, parecer*; obligatory *que* in "it seems THAT..." → **L61** *gustar* mind-flip with point/line-past → **L63** mid-translation drill (*le gusta a ella*) → **L66** *me duele la pierna* (back-to-front) → **L70** *faltar/falta*.

### 22. Numbers, time, and dates

L25 numbers 6–10 with month mnemonics (*siete/septiembre*) → **L52** days of week from planets → **L60** *a menudo* → **L64** *hace* + time = "ago" → **L65** *vez/otra vez/una vez* → **L68** *cuántas veces*.

## How the course is layered (one-paragraph summary)

The course is built like a Russian doll. **L1–L9** hand you ~1,000 verbs through convertible endings and one helper verb (*quiero*). **L10–L18** crack open the regular verb system and the stem-change rule (e→ie, o→ue). **L19–L23** add the have-past with *haber*. **L24–L28** layer in object pronouns, gender and adjective agreement. **L29–L35** introduce the *ser/estar* split and the gerund. **L36–L45** open up future, conditional, prepositions, and gustar-style verbs. **L46–L55** finish the past system (line-past + irregular participles). **L56–L68** drill point-past, fold in possessives and demonstratives, and lay down the written-accent rules. **L69–L83** install the mood tense (subjunctive) brick by brick. **L84–L88** clean up irregularities (go-verbs, irregular point-past, past mood). **L89–L90** are a rule-discovery puzzle and a closing tour of dialects.

Each new layer leans on the layer below it — almost no rule is taught from scratch. That's why the per-lesson cross-reference sections inside `lesson_NN/rules.md` so often start with "Builds on..." or "Recalls...".
