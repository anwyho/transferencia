# Lesson Bundles

The 90 lessons are grouped into **27 bundles**, one per letter of the Spanish alphabet (A–Z + Ñ). Boundaries follow the theme threads in [`cross-references.md`](cross-references.md).

Distribution: **9 bundles of 4 lessons + 18 bundles of 3 lessons = 90**. Each bundle file lives at `cards/<letter>_<theme>.yml` and aggregates every primary, extended, and cross-lesson card for the lessons it owns.

Bundle Ñ's filename uses the ASCII alias **`nn_`**: `cards/nn_line_past_full.yml`. This makes the file sort lexicographically between `n_…` and `o_…` (the literal `ñ` codepoint sorts after `z`, which broke `ls cards/` order). The bundle metadata inside the file still says `bundle: Ñ`, and the Anki deck name still renders as *Bundle Ñ Line-Past Full*. Parser regex: `^(nn|[a-zñ])_.+\.yml$`.

Cards live at `cards/<letter>_<theme>.yml`. Each file has frontmatter:

```yaml
bundle: A
title: "Foundation: vowels, es / no es, convertible-word rules"
lessons: [1, 2, 3]
cards: [...]
```

Cards inside reference the lessons they exercise via `lessons: [N1, ..., Nk]`. The card's *latest* lesson must fall inside the bundle's range (so each card has a clear "home"); earlier lessons can reference prior bundles.

## Why these boundaries

Each bundle wraps either one tight thematic unit or one accumulating thread. Notes on close calls:

- **B (4 lessons)** is the big verb-unlock arc. *-ación → -ar* (L4) and the first spine verbs / first object pronouns / first motion-with-*a* (L5–7) belong together — splitting them buries the unlock moment.
- **E (4 lessons)** consolidates the spine roll-call (L14) and the full e→ie / o→ue split arc (L15–17). The "*we* form moves stress off the root" rule lands with the splits, so L17 belongs with E rather than with F.
- **H (4 lessons)** absorbs *dar* + indirect pronouns + numbers + first adjective placement (L24–27) — the cluster that builds the "give to me" infrastructure before full agreement opens at L28.
- **M (4 lessons)** combines the two preposition swaps (*enamorarse de, pensar en*) with the conditional and *gustar*. The conditional and *gustar* both depend on the post-preposition pronoun rules (*me/te/le*), so they run together.
- **O (4 lessons)** = the *para vs por* arc: L50 introduces the split, L51 puts *estar* on top of it, L52 adds *para = in order to*, L53 closes with *podía vs podría* and the pluperfect (the same little -ía sound pulling double duty).
- **P (4 lessons)** closes out the participle system: irregular *-ado/-ido* (L54–55), relatives (L56), and the surviving *-emos/-imos* split (L57). After P, point-past starts.
- **S (4 lessons)** is the past-finishing-touches + possessives + accents block. *hace + time = ago* (L64) is the last point-past mechanic; possessives and accents (L65–67) clean up before demonstratives + mood at T.
- **V (4 lessons)** is the mood expansion arc: *elegir/para que* (L74), *donde/como/cuando + mood* (L75), *hay/había + mood* (L76), and *es + adj + que + mood* (L77).
- **Y (4 lessons)** is the go-verbs cleanup: family (L84), commands (L85), future contractions (L86), and irregular line + point-past (L87) — all variants of "the same handful of irregulars showing up everywhere."

## Ñ as a bundle letter

Yes, the Ñ slot is real. The Spanish alphabet has 27 letters: A B C D E F G H I J K L M N **Ñ** O P Q R S T U V W X Y Z. Bundle Ñ sits between N and O — same as in a dictionary.
