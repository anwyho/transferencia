# Lesson 68

## Summary
We learn the words for *this, that, these, those* (including a special "neuter" *esto* for vague "this"), plus a few useful words: *tan* (so), *cuánto/cuántas* (how much/many), *día* (day, masculine), and *creer* (to believe).

## Rules
1. *Esta noche* = "this night" = tonight. Use it as a hook to figure out the rest of the *this/that* family.
2. *Noche* is feminine, so we use *esta*. Drop the *t* and you get *esa* (that): *esa noche* = that night. Plurals: *estas noches, esas noches*.
3. For masculine, swap the ending: *este hombre* (this man), *ese hombre* (that man). Plural: *estos hombres, esos hombres*.
4. *Este* uses the /e/ from *el*, and *estos* uses the *-os* from *los* — that's how you remember the masculine forms.
5. You can also flip the order: *el hombre este, el hombre ese* = this man, that man. (Possibly an Arabic-influenced structure.)
6. NEUTER "this/that": when you don't mean a specific noun (just "this" or "that" in the air, like "I can't believe this"), use *esto* / *eso*. Not masculine, not feminine. Examples: *No creo esto.* / *Nada me gusta más que esto.*
7. *Creer* (with two e's) = to believe.
8. *Tan* = so (in the sense of "so tired"). *Tan cansado / tan cansada.*
9. Adjectives still match gender and number: *estaba tan cansado* (m) / *estaba tan cansada* (f) / *estábamos tan cansados* (m plural).
10. *Que* (no accent) = that (the joining word, not a question).
11. *Día* = day. It ends in *-a* but it's MASCULINE: *el día, todo el día* = the whole day / all day.
12. *Cuándo* = when (question), *cuánto* = how much (question — note the *t*).
13. *Cuántas veces* = how many times. Because *vez* is feminine (think *otra vez*), it's *cuántas* (feminine plural), not *cuántos*.
14. To figure out the gender of a tricky word like *vez*, look at how you used it before — *otra vez* (not *otro vez*) tells you it's feminine.
15. The line-in-past *estábamos* / *estaba* gives the past context, so verbs after it that look ambiguous (like *llegamos, dormimos*) are understood as past too.

## Vocabulary
- *esta noche* — tonight (this night)
- *este / esta / estos / estas* — this / these (m / f / m pl / f pl)
- *ese / esa / esos / esas* — that / those (m / f / m pl / f pl)
- *esto / eso* — this / that (neuter, no specific noun)
- *hombre* — man (masculine)
- *creer* — to believe
- *gustar* — to please / be liked
- *tan* — so (e.g., so tired)
- *cansado / cansada* — tired (m/f)
- *que* — that (linker)
- *día* — day (masculine despite ending in -a)
- *todo el día* — all day
- *cuándo* — when (question)
- *cuánto / cuánta* — how much (question)
- *cuántos / cuántas* — how many (question)
- *vez / veces* — time/occasion / times (feminine)
- *visitar* — to visit
- *dormir* — to sleep

## Examples / Sentences
- *Esta noche / esa noche / esas noches.* — This night / that night / those nights.
- *Este hombre / ese hombre / estos hombres / esos hombres.* — This man / that man / these men / those men.
- *El hombre este / el hombre ese.* — Same idea, flipped order.
- *No creo esto.* — I don't believe this.
- *Nada me gusta más que esto.* — I don't like anything more than this.
- *Nada me gustó más que eso.* — Nothing pleased me more than that.
- *Tan cansado / tan cansada.* — So tired (m/f).
- *Estaba tan cansada.* — I was so tired.
- *Estábamos tan cansados que dormimos todo el día.* — We were so tired that we slept all day.
- *Estábamos tan cansados cuando llegamos que dormimos todo el día.* — We were so tired when we arrived that we slept all day.
- *No sé cuántas veces lo visité.* — I don't know how many times I visited him.
- *No sé cuántas veces lo visitaron / la visitaron.* — I don't know how many times they visited him / her.
