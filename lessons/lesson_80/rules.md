# Lesson 80

## Summary
A roundup of all the uses of *se* — the self-*se*, the les→se phonetic switch, and a new "on me / on you" *se me / se te* structure for things that happen *to* you.

## Rules
1. **Verb *sé*** (with written accent) = "I know" (from *saber*). Often heard as *lo sé*, *yo sé* because the bare *sé* sounds tiny.
2. **Grammatical *se* (no accent)** = self: himself, herself, yourself (formal), itself, themselves, oneself.
3. *Se queda* = he stays / he remains himself.
4. **Phonetic *se***: when *le/les* would sit next to *lo/la/los/las*, the *le/les* changes to *se* (Spanish doesn't like *les lo*). *Le lo preparo* → *Se lo preparo* = I prepare it for him.
5. **The "on me" *se* + me/te/le/nos/les***: combine *se* with an indirect-object pronoun to express that something happened *to* you / *on* you. Common in English with only a few verbs ("it died on me", "it ran out on me"). Spanish uses it widely, with most verbs.
6. *La puerta se cierra* = the door closes itself / closes by itself.
7. *La puerta se me cierra* / *Se me cierra la puerta* = the door closes on me. Both word orders are fine.
8. *Cerrar* = to close. *La puerta* = the door.
9. *Cenar* = to dine / have dinner. *La cena* = dinner (noun).
10. *Quemar* = to burn. *La cena se quema* = the dinner gets burned. *La cena se me quema* = the dinner is burning on me.
11. *Acabar* = to finish / to run out.
12. *La batería se acaba* = the battery runs out. *La batería se me acaba* = the battery runs out on me.
13. *Siempre* = always. *La batería siempre se me acaba* = the battery always runs out on me.
14. *¿La batería se te acaba?* = does the battery run out on you? (informal)
15. *¿La batería se le acaba?* = does the battery run out on you? (formal — *le* covers all senses of "to/on/from/for you" formally).
16. Plural ("on them" or "you all"): *La batería se les acaba.*
17. **Past works the same way**. *Quemó* = it burned (point in the past). *La cena se quemó* = the dinner got burned. *La cena se me quemó* = the dinner got burned on me. *La cena se nos quemó* = the dinner burned on us.
18. You don't need to use this structure to be understood — but you'll hear it everywhere, so recognize it.

## Vocabulary
- *sé* — I know (from saber)
- *lo sé* / *yo sé* — I know (it)
- *saber* — to know (a fact)
- *se* — self (or les→se phonetic change, or "on me" structure)
- *quedar* — to stay / remain
- *se queda* — he stays
- *preparar* — to prepare
- *preparo / lo preparo / se lo preparo* — I prepare / I prepare it / I prepare it for him
- *puerta* — door (related to "port" → puerto → puerta)
- *cerrar* — to close
- *se cierra* — closes itself
- *se me cierra* — closes on me
- *cenar* — to dine / have dinner
- *cena* — dinner (noun)
- *quemar* — to burn
- *quemó* — it burned (point past)
- *acabar* — to finish / run out
- *batería* — battery
- *siempre* — always
- *me / te / le / nos / les* — indirect-object pronouns
- *se me / se te / se le / se nos / se les* — "on me / on you / on him / on us / on them"

## Examples / Sentences
- *Se queda.* — He stays.
- *Se lo preparo.* — I prepare it for him. (les+lo → se+lo)
- *La puerta se cierra.* — The door closes (itself).
- *La puerta se me cierra.* / *Se me cierra la puerta.* — The door closes on me.
- *La cena se quema.* — The dinner gets burned.
- *La cena se me quema.* — The dinner is burning on me.
- *La batería se acaba.* — The battery runs out.
- *La batería siempre se me acaba.* — My battery always runs out on me.
- *¿La batería se te acaba?* — Does the battery run out on you? (informal)
- *¿La batería se le acaba?* — Does the battery run out on you? (formal)
- *La batería se les acaba.* — The battery runs out on them / you all.
- *La cena se quemó.* — The dinner got burned.
- *La cena se me quemó.* — The dinner got burned on me.
- *La cena se nos quemó.* — The dinner got burned on us.
