# Lesson 52

## Summary
*Para* also means "in order to" — and we MUST translate "to" as *para* whenever it implies "in order to". We also build the *-ing* form (*-ando/-iendo*) used with *estar*, and learn how *-ing* English adjectives become *-ante/-iente* in Spanish.

## Rules
1. *Para* = "in order to" as well as "for". Whenever English "to" really means "in order to", you must use *para* in Spanish.
2. *Hemos hablado para resolverlo* = "We have spoken (in order) to resolve it". You can't say *hemos hablado a resolverlo* or *hemos hablado resolverlo* — you need *para*.
3. The "to" form alone in Spanish doesn't carry the "in order to" meaning. Add *para* to get it.
4. *Estoy aquí para verte* = "I am here (in order) to see you". Negative-language-transfer clue: Spanish speakers sometimes say "I am here for see you" in English — that mistake came FROM Spanish.
5. After all prepositions (*para, por, de, en, con, a*) we use the "to" form, NOT *-ing*. *Te he llamado para hablarlo* = "I have called you (in order) to speak about it" (no "about" needed — *hablarlo* covers it).
6. To make the *-ing* form (only used WITH *estar* for "I am doing X right now"): chop *-ar* and add *-ando*; chop *-er/-ir* and add *-iendo*. *Comer* → *comiendo*; *hablar* → *hablando*.
7. *Estoy + -ando/-iendo* = I am ___ing right now. *Estoy comiendo* = I'm eating. *Estoy hablando* = I'm speaking.
8. Object pronouns can stick on the END of the *-ando/-iendo* form, OR sit before *estar*. Both work: *Estoy explicándolo* = *Lo estoy explicando* = I'm explaining it.
9. *Explicación* (NOT *explanación*) = explanation. The /k/ sound stays from Latin (*plicare* = to fold). *Explicar* = to explain (= to unfold).
10. *Estaba + -ando/-iendo* = I was ___ing. So you have TWO ways to say "I was eating": *comía* (line-past) OR *estaba comiendo*. Mostly interchangeable; you'll pick up which fits better.
11. *Tomar* = to take. *Estoy tomando un curso de español* = I'm taking a Spanish course (even if not at this exact moment).
12. *Curso* (with *u*, not *a*) = course.
13. *Estaba tomando un curso de español* / *Tomaba un curso de español* — both = I was taking a Spanish course.
14. We DON'T use *-ando/-iendo* after prepositions or to make general statements. "Swimming is healthy" = *nadar es saludable* (use the "to" form), NOT *nadando*.
15. *Salud* = health (the toast); *saludable* = healthy.
16. To make English *-ing* adjectives in Spanish: chop the verb ending and add *-ante* (for *-ar* verbs) or *-iente* (for *-er/-ir* verbs).
17. Examples: *interesar* → *interesante* (interesting). *Irritar* → *irritante* (irritating). *Frustrar* → *frustrante* (frustrating). *Menguar* → *menguante* (waning). *Salir* → *saliente* (salient/standing out). *Vivir* → *viviente* (living).
18. *La luna menguante* = the waning moon (adjective uses *-ante*, NOT verb form *menguando*).
19. Spanish day-of-week names come from planets (very visible compared to English): *lunes* (luna/moon), *martes* (Mars), *miércoles* (Mercury), *jueves* (Jupiter, with j), *viernes* (Venus).
20. *Ser* used as a noun = "a being". *Un ser humano* = a human being. *Un ser viviente* = a living being.
21. *Vivo* = alive (adjective from *vivir*).

- *para* — for / in order to
- *resolver* — to resolve (*lo resolverlo* — to resolve it)
- *intentar* — to try (related to "intent")
- *hablar / hablarlo* — to speak / to speak about it
- *llamar / te he llamado* — to call / I have called you
- *estar / estoy / estaba* — to be (states)
- *-ando / -iendo* — -ing endings (for -ar / -er-ir)
- *comer / comiendo* — to eat / eating
- *hablar / hablando* — to speak / speaking
- *explicar / explicando / explicación* — to explain / explaining / explanation
- *tomar* — to take
- *un curso (de español)* — a (Spanish) course
- *salud / saludable* — health / healthy
- *nadar* — to swim
- *bailar* — to dance
- *interesar / interesante* — to interest / interesting
- *irritar / irritante* — to irritate / irritating
- *frustrar / frustrante* — to frustrate / frustrating
- *menguar / menguante* — to wane / waning
- *la luna* — the moon
- *lunes, martes, miércoles, jueves, viernes* — Mon-Fri (planet names)
- *vivir / vivo / viviente* — to live / alive / living
- *ser* — to be / a being
- *un ser humano / un ser viviente* — a human being / a living being
- *salir / saliente* — to leave / salient

## Examples / Sentences
- *Hemos hablado para resolverlo* — We have spoken (in order) to resolve it
- *Hemos hablado para intentar resolverlo* — We have spoken (in order) to try to resolve it
- *Estoy aquí para verte* — I am here (in order) to see you
- *Te he llamado para hablarlo* — I've called you (in order) to speak about it
- *Estoy comiendo* — I'm eating (right now)
- *Estoy hablando* — I'm speaking (right now)
- *Estoy explicándolo / Lo estoy explicando* — I'm explaining it
- *Estoy tomando un curso de español* — I'm taking a Spanish course
- *Estaba tomando un curso de español* — I was taking a Spanish course
- *Tomaba un curso de español* — I was taking a Spanish course
- *Comía / Estaba comiendo* — I was eating (both work)
- *Bailar es saludable* — Dancing is healthy
- *Nadar es saludable* — Swimming is healthy

- *Es irritante* — It is irritating
- *Frustrante* — Frustrating
- *La luna menguante* — The waning moon

- *Un ser viviente* — A living being
