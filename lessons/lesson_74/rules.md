# Lesson 74

## Summary
More mood-tense practice with new verbs (*elegir*, *romper*, *encontrar*, *volver*), plus the rules for *por mí* vs *para*, *para que*, and the tiny word *también*.

## Rules
1. *Elegir* = to choose (think "elect"; *elección* = election).
2. **e → i shift when stressed**: in some verbs the *e* doesn't split into *ie*, it just turns into *i* when stressed. *Elegir* → *elijo, eliges, elige, elegimos, eligen* — the *e* becomes *i* in every form except *nosotros* (*elegimos*) where the stress is elsewhere.
3. **Spelling note**: *elegir* → *elijo* (g changes to j before o/a to keep the soft sound).
4. **"For me" — *por mí* vs *para mí***:
   - *por mí* = on my behalf / in my place. *No quiero que elijas por mí* = I don't want you to choose on my behalf.
   - *para mí* = literally for me (something destined for me).
5. *Romper* = to break (think "rupture", "disrupt", "erupt"). *Romperse* = to break (without saying who broke it).
6. *No quiero que se rompa* = I don't want it to break (no blame on anyone). *No quiero que lo rompa* would mean "I don't want him/her/you to break it."
7. **para = "in order to"**: when English "to" really means "in order to", use *para*. *Hemos cocinado para comer* = we've cooked (in order) to eat.
8. **para que + mood tense**: *para que* = "so that" / "in order that". The verb after must go to the mood tense. *Hemos cocinado para que coman* = we've cooked so they eat.
9. You can't shortcut *para que coman* into *para comer* — that would mean "in order for us to eat." Use *para que* whenever the doer is different.
10. *Volver* = to return / to go back / to come back (think "revolve"; in Spanish *revolver* means to stir).
11. *Volver* is an **o → ue stem-changer**: *vuelvo, vuelves, vuelve, volvemos, vuelven*. The *o* splits into *ue* when stressed.
12. **Mood-tense of *volver*** keeps the same split: *vuelva, vuelvas, vuelva, volvamos, vuelvan*.
13. *Hecho* = the irregular past participle of *hacer* (after *haber*). *He hecho* = I have done. *Hecho* by itself can also mean "fact".
14. **Dialect tip**: *para* often shortens to *pa* in casual speech. *Pa que vuelvas* = *para que vuelvas*.
15. *Tan* = so (so tired = *tan cansado*).
16. *También* = also / as well — actually has *tan* hidden inside (*tan + bien*, with n softening to m before b). Mirrors English "as well" exactly.

## Vocabulary
- *elegir* — to choose (related to elect)
- *elijo / eliges / elige / elegimos / eligen* — choose forms
- *elección* — election
- *por mí* — on my behalf
- *para mí* — for me (literally for me)
- *romper* — to break
- *romperse* — to break (without naming a doer)
- *encontrar* — to find
- *encontrarse* — to find oneself / to meet up
- *volver* — to return / come back / go back
- *vuelvo / vuelves / vuelve / volvemos / vuelven* — return forms
- *vuelva / vuelvas / vuelva / volvamos / vuelvan* — mood forms of volver
- *revolver* (Spanish) — to stir
- *hacer* — to do / make
- *hecho* — done (after haber); also "fact"
- *cocinar* — to cook
- *para* — for / in order to
- *para que* — so that / in order that (+ mood tense)
- *pa* — casual short form of *para*
- *tan* — so
- *también* — also / as well

## Examples / Sentences
- *No quiero que elijas por mí.* — I don't want you to choose for me / on my behalf.
- *No quiero que se rompa.* — I don't want it to break.
- *No quiero que lo rompa.* — I don't want him to break it.
- *Nos hemos encontrado para hablar.* — We've met up (in order) to speak.
- *Hemos cocinado para que coman.* — We've cooked so that they eat.
- *Vuelvo. / Vuelves.* — I come back. / You come back.
- *Para que vuelvas.* — So that you come back.
- *Lo he hecho para que vuelvas.* — I've done it so that you come back.
- *Lo han hecho para que volvamos.* — They've done it so that we come back.
- *Para que tú también vuelvas.* — So that you too come back.
- *Hemos hecho todo para que tú también vuelvas.* — We've done everything so that you also come back.

## Cross-references
- Builds on prior rules: *para* for "in order to", and the irregular *hecho* after *haber*.
- Builds on prior stem-change pattern (e → ie, o → ue) — here introduces a third pattern (e → i).
- Recalls reflexive *-se* used to leave the doer unnamed (*romperse*, *encontrarse*).
