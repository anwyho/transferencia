# Lesson 05

## Summary
We meet *yo* ("I"), the reflexive *me* ("myself"), and the object pronoun *lo* ("it/him"). We learn the rolled *r* (it likes a relaxed mouth) and a bonus rule: English `-ence`/`-ance` → Spanish `-encia`/`-ancia`.

## Rules
1. ***Yo* = "I", but you only use it for emphasis.*** *Quiero* alone = "I want." *Yo quiero* = "*I* want" (with stress on me).
2. **Word order matches English: *yo no quiero* = "I don't want."** Not *no yo quiero.*
3. ***Te* is always informal "you."** Used with friends, family, etc.
4. ***Me* = "me" / "myself"** — same as English, just pronounced /meh/. Stick it on the end of the verb: *quiero prepararme* = "I want to prepare myself" / "I want to get ready."
5. **Build sentences block by block.** "I don't want to oblige you" feels impossible until you split it: *no quiero* + *obligar* + *te* → *no quiero obligarte.* Pronouns hook onto the end of the "to ___" verb.
6. ***Lo* = "it" or "him."*** Stick it on the end of the verb: *quiero cancelarlo* (I want to cancel it), *quiero salvarlo* (I want to save him).
7. **The rolled *r*: needs a *relaxed* mouth, not a tense one.** Lift the tongue gently and let air flap it. If you can't do it, don't stress — even native speakers struggle. The more you hear it, the more it'll come.
8. **A single *r* between vowels = a soft tap (like English "butter" /budder/). A double *rr* or an initial *r* = rolled.** *pero* (but) vs *perro* (dog). *real* and *regular* roll because they start with *r*.
9. ***h* before vowels in Spanish is sometimes silent later** (we'll meet that), but for now: word-initial *r* rolls.
10. **Bonus rule: English `-ence`/`-ance` → Spanish `-encia`/`-ancia`.** *experience → experiencia, difference → diferencia, influence → influencia, existence → existencia, consistency → consistencia, incongruence → incongruencia.*
11. ***Experimentar* means both "to experiment" *and* "to experience."*** Built-in wisdom: your experience should be like an experiment — observe, learn. The nouns split (*experiencia* vs *experimento*), but the verb is one.
12. **Take new words *syllable by syllable* — don't let it become "a salad."*** *Participación → par-ti-ci-pa-CIÓN → participar.* If you rush, you'll add extra sounds like "participatar."
13. **For words with no obvious -ation form, go the long way through the noun anyway.** *Creation → creación → crear* (to create).

## Examples / Sentences
- *Yo quiero.* — I want (with emphasis).
- *Yo no quiero.* — I don't want.
- *Yo no quiero cancelar.* — I don't want to cancel.
- *Quiero celebrar.* — I want to celebrate.
- *Quiero ilustrar.* — I want to illustrate.
- *Quiero informarme.* — I want to find out (literally "inform myself").
- *No quiero obligarte.* — I don't want to oblige you / make you do it.
- *No quiero provocarte.* — I don't want to provoke you.
- *Quiero cancelarlo.* — I want to cancel it.
- *Quiero regularlo.* — I want to regulate it.
- *Quiero salvarlo.* — I want to save him / it.
- *Quiero prepararme.* — I want to get ready.
