# Lesson 16

## Summary
More vowel-splitting practice: we drill *empezar* (to start) where *e → ie*, and we meet a bunch of *o → ue* splitters: *poder → puedo*, *dormir → duermo*, *encontrar → encuentro*, *mostrar → muestro*. We also see how to spot the noun version of a verb (the one with *un / una / el / la* in front).

## Rules
1. *Comenzar* = to commence / to start (cousin of "commence"). *Empezar* is more common — same meaning.
2. *Empezar*: stress on penultimate syllable in present makes the *e* split → *empiezo* (I start), *empieza* (he/she), *empiezan* (they), *empiezas* (you-inf).
3. CRITICAL HABIT: BEFORE you change a verb, run back to the "to" form. Otherwise you'll guess and get vowel harmony nonsense.
4. *El* = the (masculine). *La* = the (feminine). If a word has *el / la / un / una* in front, it's a noun, even if it looks like a verb. *El poder* = the power (noun). *Poder* alone (no the) = to be able (verb).
5. *Poder* = to be able / "to can" (English doesn't have a "to can," so we use "to be able"). It's related to the word *power*.
6. *Poder → puedo* (I can): the *o* splits into *ue* under stress. Same with the e → ie pattern.
7. You can see this splitting between English and Spanish words too: port → *puerto* (Puerto Rico = "the rich port"; *rico* = rich/nice/tasty); force → *fuerza*; school → *escuela*; corp(orate) → *cuerpo* (body); innov- → *nuevo* (new); pop(ulation) → *pueblo* (village/people; population is *población*, exception); mort- → *muerte* (death; mortgage = "death pact").
8. Adjectives match nouns in gender/number: *muerto / muerta / muertos / muertas* (dead). The noun version takes "the": *la muerte* (death).
9. *Dormir* (to sleep; cf. dormant volcano, dormitory) → *duermo* (I sleep), *duerme* (he/she), *duermen* (they). The *o* splits.
10. To say "I fall asleep," use the reflexive: *me duermo* (literally "I sleep myself"). This is a very common pattern.
11. *Encontrar* = to find (also can mean to encounter / to meet). *Encuentro* = I find. The verb form *encuentro* and the noun *un encuentro* (an encounter) look the same — context (*un / el*) tells you it's a noun.
12. Many noun forms hide as verb forms. If you see *un / una / el / la* in front of it, it's a noun.
13. *Demostrar* (to demonstrate). Strip *de-* (which means "of") and you get *mostrar* (to show). *Mostrar → muestro* (I show), *muestra* (he/she shows), *muestran* (they). *Una muestra* (a sample) is the noun.
14. Don't roll the *r* in *muestra* — it's just one *r*. Roll it only with *rr* or word-initial *r*.

## Vocabulary
- *comenzar* — to commence / start
- *empezar* — to start (more common)
- *empiezo / empieza / empiezan / empiezas* — start (forms)
- *poder* — to be able / can; *el poder* — the power (noun)
- *puedo* — I can
- *puerto* — port (cf. Puerto Rico = "the rich port")
- *rico* — rich / nice / tasty / sweet
- *fuerza* — force
- *escuela* — school
- *cuerpo* — body (cf. corp-)
- *nuevo* — new (cf. innov-)
- *pueblo* — village / a people (cf. pop-)
- *población* — population (exception, no split)
- *muerte* — death (la); *muerto/a* — dead
- *dormir* — to sleep (cf. dormant, dormitory)
- *duermo / duerme / duermen* — sleep forms
- *me duermo* — I fall asleep
- *encontrar* — to find / meet / encounter
- *encuentro* — I find (verb); *un encuentro* — an encounter (noun)
- *demostrar* — to demonstrate
- *mostrar* — to show
- *muestro / muestra / muestran* — show forms; *una muestra* — a sample (noun)
- *de-* — of (Spanish prefix)
