# Lesson 77

## Summary
The mood tense fires up after "it is + adjective + que" (it's terrible/possible/important/necessary that...) and after exclamations like *qué fuerte que...*

## Rules
1. *Debería* = should (literally "would owe / would must" from *deber*).
2. *Debería haber algo para comer* = there should be something to eat (and *haber* = exist, not ser/estar).
3. **Quick check for "to be" = exist**: if neither ser nor estar fits, it's *haber*.
4. **It is + adjective + *que* + verb → mood tense.** *Que* triggers the mood tense in many of these spots.
5. *Es terrible que comas esto* = it's terrible that you eat this (mood tense after *que*).
6. *Es terrible si comes esto* = it's terrible if you eat this (no mood tense — *si* doesn't trigger it here).
7. *Es posible que vayamos más tarde* = it's possible that we (might) go later.
8. *Más tarde* = later.
9. *Fuerte* = strong, but colloquially also "hardcore / heavy / full-on". *¡Qué fuerte!* = wow, that's heavy/hardcore.
10. ***Qué + adjective*** = how + adjective. *Qué fuerte* = how heavy. *Qué lindo* = how nice. *Qué bueno* = how good.
11. ***Qué + adjective + que + verb (mood tense)*** = how [adjective] that... *¡Qué fuerte que esté aquí!* = how heavy that he's here.
12. English uses words like *should* or *might* to give a similar mood-tense flavor (e.g., "how incredible that he should come").
13. *Está bien* = literally "it's well", but also used to mean "it's okay".
14. *Está bien que preguntes* = it's okay that you (might/may) ask.
15. *Preguntar* = to ask. *Responder* = to reply / respond.
16. *No es necesario que responda* = it's not necessary that I respond. (Mood tense after *no es necesario que*.)
17. Equivalent: *No hace falta que responda* = it doesn't need to / it doesn't make lack that I respond.
18. *Es importante que lo encontremos pronto* = it's important that we find it soon.

## Vocabulary
- *deber* — to owe / must
- *debería* — should
- *haber* — to exist (the "be" of "there is")
- *algo* — something
- *para comer* — to eat / in order to eat
- *terrible* — terrible
- *si* — if (no mood tense)
- *posible* — possible
- *más tarde* — later
- *vaya / vayamos* — mood form of ir
- *fuerte* — strong / hardcore / heavy (colloquial)
- *qué* — how / what
- *lindo* — nice / pretty
- *bueno* — good
- *increíble* — incredible
- *está bien* — it's okay / it's well
- *preguntar* — to ask
- *responder* — to reply / respond
- *necesario* — necessary
- *necesidad* — necessity
- *no hace falta* — it's not necessary / it doesn't make lack
- *importante* — important
- *encontrar* — to find
- *encontremos* — (mood) we find
- *pronto* — soon

## Examples / Sentences
- *Debería haber algo para comer.* — There should be something to eat.
- *Es terrible que comas esto.* — It's terrible that you eat this.
- *Es terrible si comes esto.* — It's terrible if you eat this.
- *Es posible que vayamos más tarde.* — It's possible we (might) go later.
- *¡Qué fuerte que esté aquí!* — How heavy/hardcore that he's here!
- *Está bien que preguntes, pero no es necesario que responda.* — It's okay that you ask, but it's not necessary for me to respond.
- *No hace falta que responda.* — I don't need to respond.
- *Es importante que lo encontremos pronto.* — It's important that we find it soon.
