# Lesson 85

## Summary
We continue with the "go verbs": their mood tense (used for formal commands and negative commands) goes to *-ga*, but for **informal positive commands** there's a shortcut — just drop the *-go*. So *salgo → sal*, *tengo → ten*, *vengo → ven*, *digo → di*, *pongo → pon*.

## Rules
1. Go-verbs go to *-ga* in the mood tense, and that mood form is also used for FORMAL commands and ALL negative commands. *Salga* = leave (formal). *Salgan* = leave (formal plural). *No salgan* = don't leave.
2. For positive commands, object pronouns get attached to the END (and we may need a written accent to keep the stress): *Lo ponga aquí* (mood) → *Póngalo aquí* (positive command, "put it here"). For negative, the pronoun goes BEFORE: *No lo ponga aquí* (don't put it here).
3. Reminder: with regular verbs, the shortcut for **informal positive commands** is "use the *he/she* form" — drop the *s* off the *you-informal* form. *Comes → come* (eat!). *Encuentras → encuéntralo* (find it!).
4. **Special shortcut for go-verbs informal positive command: just drop the *-go* off the *I*-form.**
   - *tengo → ten* (have!) — *Tenlo* (have it!)
   - *salgo → sal* (leave!)
   - *vengo → ven* (come!)
   - *digo → di* (say! / tell!) — *Dime* (tell me!)
   - *pongo → pon* (put!)
5. Phone-answer mnemonic: in Spain you hear *dígame* (formal "tell me") and *dime* (informal "tell me"). Both are commands attached to *me*. *Dígame* = formal mood + me; *dime* = informal shortcut (drop *-go* of *digo* → *di*) + me.
6. Heads-up about *venga* in Spain: as "come on!" it's lost its formality — it's a fixed expression. So *ven* sounds more like physical "come (here)" while *venga, venga* is used like "come on, come on."
7. After *cuando* (when), if we're talking about the future, we use the mood tense. *Dime cuando vengas* = "Tell me when you come" (= when you arrive in the future). *Dígame cuando lo haga* = "Tell me when you do it" (formal).
8. So the two related shortcuts: for go-verbs, mood tense = drop *-go*, add *-ga*-endings; informal positive command = drop the *-go* and stop there.

## Vocabulary
- *salir / salgo / salga / salgan / sal* — to leave / I leave / leave! (formal sing.) / leave! (formal pl.) / leave! (informal)
- *poner / pongo / ponga / póngalo / pon* — to put / I put / put! (formal) / put it! (formal pos. cmd) / put! (informal)
- *tener / tengo / tenga / téngalo / ten / tenlo* — to have, plus its commands
- *venir / vengo / venga / ven* — to come, plus its commands
- *decir / digo / diga / dígame / di / dime* — to say/tell, plus its commands
- *hacer / haga* — to make/do (mood form)
- *aquí* — here
- *cuando* — when
- *esta noche* — tonight (implied carry-over)

## Examples / Sentences
- *Lo ponga aquí.* — (Mood) that you put it here.
- *Póngalo aquí.* — Put it here. (formal positive command)
- *No lo ponga aquí.* — Don't put it here.
- *No salgan.* — Don't leave (you-all formal).
- *Tenlo.* — Have it. (informal command)
- *Téngalo.* — Have it. (formal command)
- *Dígame.* — Tell me. (formal phone)
- *Dime.* — Tell me. (informal phone)
- *Sal.* — Leave! (informal)
- *Ven.* — Come! (informal)
- *Dime cuando vengas.* — Tell me when you come.
- *Dígame cuando lo haga.* — Tell me when you do it. (formal)
